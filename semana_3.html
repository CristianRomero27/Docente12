Semana 3

💡Tema:

Diseño y recorte de materiales para la construcción del Escape Room.

 

🎯 Objetivo:

Diseñar y cortar materiales para construir el Escape Room.

 

👩🏻‍💻 Desarrollo de competencias:

  • Desarrolla habilidades técnicas para manipular materiales de forma segura y precisa.
  • Aplica conocimientos previos en diseño y planificación para la elaboración de elementos funcionales.
  • Colabora en equipo para construir un entorno lúdico basado en una narrativa.

📌 Requisitos previos:

Requisitos previos:

  • Conocimiento básico de diseño estructural.
  • Familiaridad con herramientas de corte (tijeras, cúter, sierras, etc.).
  • Habilidad para trabajar en equipo y seguir instrucciones técnicas.

🌍 Contexto:

  • Contexto educativo: taller escolar o proyecto interdisciplinario.
  • Contexto tecnológico: aplicación de herramientas y materiales en la creación de escenarios interactivos.
  • Contexto colaborativo: integración de ideas en grupo para la construcción conjunta del Escape Room.
